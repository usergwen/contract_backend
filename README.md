# contract_backend
契约测试管理平台后端项目源码
> 说明:因为一些原因,删除了部分功能和源代码,只保留了最核心的功能

## 1. 技术栈
* Python3.8
* fastapi

## 2. 功能介绍
* 推送契约文件到 pact broker
* 自动获取微服务 api 进行契约验证

## 3. 相关文档
* pact-python: https://github.com/pact-foundation/pact-python
* pact-broker: https://github.com/pact-foundation/pact_broker
